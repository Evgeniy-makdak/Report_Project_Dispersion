# Проект «ДИСПЕРСИЯ»

> **Космический эксперимент по исследованию активных дисперсных сред в условиях микрогравитации**

---

## 📋 О проекте

Проект **«Дисперсия»** — это уникальный космический эксперимент, реализованный на борту Международной космической станции (МКС) в 2022–2023 годах. Цель эксперимента — изучение поведения сложных материалов (полимеров, суспензий, эмульсий и «умных» жидкостей) в условиях невесомости.

На Земле гравитация искажает естественные физические процессы: тяжёлые частицы оседают, лёгкие всплывают, возникают конвекционные потоки. В космосе, в отсутствие гравитации, можно увидеть, как ведут себя материалы «в чистом виде» — без этих помех.

## 🎯 Задачи эксперимента

1. **Фазовый распад полимер — растворитель**
   - Изучение расслоения смесей поливинилметилового эфира, поливинилкарбазола и поливинилпирролидона с водой при изменении температуры.

2. **Дисперсные системы**
   - Формирование, стабилизация и разрушение структур из микро- и наночастиц (стеклосферы, полистирол, фуллеренсодержащая сажа).

3. **Электро- и магнитореологические жидкости**
   - Управление структурой «умных» материалов с помощью электрических и магнитных полей.

## 🛰️ Научная аппаратура

Прибор разработан **АО «Лазерные системы»** (Санкт-Петербург) и включает:

- **3 лазера** (400–700 нм) для дифракционных измерений
- **Виброопору** (5–20 Гц, амплитуда до 1 мм) для перемешивания образцов
- **Систему нагрева** (20–80 °C) для исследования фазовых переходов
- **Блоки электрического** (до 700 В/м) и **магнитного** (до 500 А/м) полей
- **Цветную видеокамеру** (1920×1200) для визуальной фиксации процессов
- **Систему управления и сбора данных (СУСД)** — полностью отечественное ПО

## 🔬 Прикладное значение

Результаты эксперимента востребованы для:
- **Программы освоения Луны** — создание строительных материалов из лунного грунта
- **Промышленности** — разработка «умных» амортизаторов, протезов, робототехники
- **Фармацевтики** — создание лекарственных форм с контролируемым высвобождением
- **Образования** — использование реальных космических данных в учебном процессе

## 🚀 Перспективы

На базе прибора «Дисперсия» разрабатываются новые эксперименты:
- **«Микс-Реголит»** (2025–2026) — вибрационное смешивание имитатора лунного грунта
- **«Пыль-УФ»** (2024–2025) — исследование плазменно-пылевых образований в открытом космосе
